# Movie Buzz! 

Welcome to `Movie Buzz`! This is the starting point of a full-stack application that we will be builiding together over the course of this class. Each week we will build upon this project, applying the new skills you will be learning along the way. Before we begin, let's cover a few things first:

## Starting Out

This directory and its contents will be your starting point for the application we're going to build. You should copy this entire `/week_01_bash_and_git/movie-buzz` directory to your `Documents` folder now. We will replace and improve parts each week as we work on this project. 

## Weekly Updates

As mentioned above, we will be adding to this project every week. If for some reason you fall behind or something happens to your project, up-to-date versions will be provided here in your RI git repo each week so you can easily catch up. 

In the event that you need a new copy, follow the regular procedure of `git pull`ing from this RI repo, then copy files from your local repo to your `Documents/movie-buzz` folder and continue from there.

## Independent Research

If you want to experiment and play with the project on your own, please do so in a separate copy to ensure your `Documents/movie-buzz` version doesn't get out of sync with the class.

And remember: Have fun! This is intended to be a social learning experience, just like you'll have at bootcamps or apprenticeships after prison: The more engaged and excited you are about the work we do here, the more you'll learn! 

We hope that you will take the concepts you practice here and develop new projects or apply them to your existing projects. Good luck!
